# High-Performance Web Development Portfolio

A simple Next.js landing page for a developer portfolio with a hero section, services, featured projects, testimonials, and contact footer.

## Scripts

- `npm install` - install dependencies
- `npm run dev` - start local development server
- `npm run build` - build for production
- `npm run start` - run production server
